# Projeto: Controle de Notas

## Objetivo Geral
Desenvolver um sistema em C para controlar notas de Livros, localizar livros rapidamente e ter um histórico confiável de empréstimos

## Principais Funcionalidades Previstas
•	Dados do livro: título, autor, ISBN (ou código), edição, ano, quantidade de exemplares.
•	Dados do leitor: nome, ID/matrícula, telefone ou e-mail.
•	Registro de empréstimo: ID do livro, ID do leitor, data de empréstimo, data prevista de devolução, status (emprestado/devolvido).
•	Histórico: lista de empréstimos por livro e por leitor.
•	Observações (opcional): estado do livro, notas sobre perdas ou danos.
